# UAS-HAMEED-RIZKIY-AMRUDDIN

PENKJELASAN :
website untuk membuat TO DO LIST  simpel yang di rangkai dengan sedikit pusing

ALUR ALGORITMA/LOGIKA :
ketika user memasukan text pada kolom isian list maka akan di proses oleh logika yang di rangkai di java script kemudian menghasilkan output to do list apa yang akan di kerjakan, dan jika sudah selesai user bisa memilih selesai atau belum, jika selesai maka akan otomatis terhapus dari web, jika belum maka selesaikan lah
